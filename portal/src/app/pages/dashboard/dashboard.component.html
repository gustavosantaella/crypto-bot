Technical Documentation

📊 RSI (Relative Strength Index)

Indicador de tipo oscilador que mide la velocidad y el cambio de los movimientos de precios. Se utiliza para identificar condiciones de sobrecompra (RSI > 70) o sobreventa (RSI < 30). El bot utiliza estas señales para anticipar reversiones de tendencia.

📉 ATR (Average True Range)

Indicador que mide la volatilidad del mercado. El bot utiliza el ATR para calcular distancias de Stop Loss y Take Profit dinámicas. Si el mercado está muy volátil, el ATR aumenta y el bot "aleja" los objetivos para evitar ser sacado por ruido de precio.

⚖️ Leverage (Apalancamiento)

Herramienta que permite operar con un capital mayor al que realmente se tiene en la billetera. Un apalancamiento de 5x significa que por cada $1 de tu balance, el bot abre una posición de $5. Esto amplifica tanto las ganancias como las posibles pérdidas.

🔄 Long vs Short

LONG: Apuesta a que el precio subirá. Se entra comprando y se sale vendiendo.
SHORT: Apuesta a que el precio bajará. Se entra vendiendo (prestado) y se sale comprando (devolviendo) a un precio menor.

💰 PnL (Profit and Loss)

El PnL Realizado es la ganancia o pérdida neta que se consolida al momento de cerrar una operación. Se calcula restando los costos de entrada de los beneficios de salida, ajustado por la dirección del trade (Long/Short).
